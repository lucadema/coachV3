# PDF font files

Place these files in this folder to embed Inter in the generated PDF:

- Inter-ExtraLight.ttf
- Inter-ExtraLightItalic.ttf
- Inter-Bold.ttf
- Inter-BoldItalic.ttf

The generator uses Inter ExtraLight as the normal/default font, matching the Word sample style.
If these files are missing, the PDF generator will show a clear error. The approved PDF layout requires Inter everywhere.
